## üìä All Features Explained

### üéØ RAGE TAB

#### **Aimbot**
- **Toggle:** Enable/disable aimbot
- **FOV:** Adjustable aim circle (10-500 pixels)
- **Smoothness:** How smooth the aim moves (1-100)
- **Target Part:** Choose Head, Torso, or Closest
- **Visibility Check:** Only aim at visible players
- **Team Check:** Avoid targeting teammates
- **Keybind:** Right Mouse Button (hold to aim)

**Usage Tips:**
- Start with FOV around 100-150 for testing
- Higher smoothness = more natural (60-80 recommended)
- Enable Visibility Check to avoid suspicion
- Use Team Check in team-based games

#### **TriggerBot**
- **Toggle:** Auto-fire when crosshair on enemy
- **Delay:** Milliseconds before shooting (1-1000ms)

**Usage Tips:**
- Set delay to 50-100ms for natural feel
- Combine with aimbot for best results
- Works great in FPS games

#### **Anti-Aim**
- **Toggle:** Enable anti-aim
- **Pitch:** Up, Down, Jitter, Random
- **Yaw:** Spin, Backwards, Jitter, Random

**Usage Tips:**
- Makes you harder to hit
- May look suspicious to other players
- Best used in chaotic situations

#### **Resolver**
- **Toggle:** Enable resolver
- **Mode:** Basic or Advanced

**Usage Tips:**
- Helps aim at anti-aiming players
- Use Advanced mode for better results

## üêõ Troubleshooting

### GUI Won't Show
1. Press `Right Shift`
2. Check if CoreGui is blocked
3. Try re-executing the script

### Aimbot Not Working
1. Ensure it's toggled ON
2. Check FOV is large enough
3. Verify players are in range
4. Enable Visibility Check

### ESP Not Visible
1. Toggle ESP on
2. Adjust colors (may blend with background)
3. Check if Box ESP is enabled
4. Ensure Drawing library is supported

### Key Invalid
1. Use exact key: `DUMMYHOOK-PREMIUM-2025`
2. Check for typos
3. Ensure caps lock is correct
4. Try generating new key with Discord bot

### Features Not Working
1. Game may have anti-cheat
2. Feature may not support that game
3. Try in different game
4. Check executor compatibility
